<h1 align="center">udict</h1>

<p align="center">
  一个离线 MDX/MDD 词典查询 uTools 插件，内置生词本。
</p>

---

## 安装

在 uTools 插件市场搜索 **udict**，点击安装即可。

## 功能特性

- **离线查词** — 直接解析本地 `.mdx` / `.mdd` 文件，完全离线
- **多词典并查** — 一次查询所有已配置词典，按来源分区展示
- **多卷 MDD** — 自动发现同名 `.mdd` / `.1.mdd` / `.2.mdd` 分卷资源
- **富媒体资源** — 发音、图片、嵌入字体、交叉引用链接全部可用
- **前缀建议** — 输入几个字母即时弹出候选词
- **Key 索引缓存** — 首次加载提取 key 索引写入磁盘，冷启动前缀建议 0ms
- **生词本** — 将生词收录到多个自定义生词本，支持 1–5 星难度与自定义备注
- **可折叠侧栏** — 词典跳转、字号调节、历史记录、生词本管理
- **Claude 风格主题** — 暖色奶白 + 珊瑚橙配色，强制浅色保证阅读一致性

## 使用方法

### 查词

- 打开插件（默认关键字 `udict`），或在 uTools 主输入框直接输入中文 / 英文，点击 `udict 查词` 匹配项进入插件。
- **Enter** 查询，**↑/↓** 选择建议，**Esc** 清空输入 / 退出插件。

### 生词本

- 点击搜索框旁边的 ☆ 收录当前生词，弹窗内可多选加入到哪些生词本。
- 侧栏 `+` 新建生词本；拖动 `≡` 句柄排序；双击名称重命名（默认生词本锁定）。
- 点击生词本进入表格视图：可按生词 / 收录时间 / 难度排序，编辑备注，下拉选择难度。
- **默认生词本** 自动汇总所有生词本中的词，附带"来源"列显示归属。

### 设置

- 点击 ⚙ 图标管理词典。拖动 `≡` 可调整词典顺序（影响查询结果显示顺序）。
- `Pick directory…` 扫描目录，自动添加目录下所有 `.mdx` 文件。
- **Esc** 返回查词页。

### 彩蛋

在搜索框输入 `udict` 可查看关于页面。
